5. Flexboxの活用

Flexboxの定義

Flexbox（Flexible Box）は、コンテナ内のアイテムのレイアウトを行うためのCSSのモジュールです。Flexboxを使用すると、コンテナ内のアイテムを柔軟に配置し、スペースの分配やアイテムのサイズ変更を容易にすることができます。

Flexboxの主なプロパティとその使い方

  1. flex-direction: Flexコンテナ内のアイテムの配置方向を指定します。デフォルト値はrowですが、row-reverse、column、column-reverseなども使えます。
  2. justify-content: メイン軸（flex-directionで指定した方向）に沿ってアイテムを配置する際の方法を指定します。flex-start、flex-end、center、space-between、space-aroundなどの値があります。
  3. align-items: クロス軸（flex-directionと直交する方向）に沿ってアイテムを配置する際の方法を指定します。flex-start、flex-end、center、baseline、stretchなどの値があります。
  4. flex: アイテムの伸縮性を設定します。flex-grow、flex-shrink、flex-basisのショートハンドです。
  5. align-self: 個々のアイテムに対してalign-itemsと同様の機能を提供します。

実際のFlexboxの活用例

```css .container { display: flex; flex-direction: row; justify-content: space-between; align-items: center; }

.item { flex: 1; } html

Item 1

Item 2
Item 3

`` この例では、.container内の.itemをFlexアイテムとして配置しています。flex-direction: rowにより、アイテムが水平方向に並びます。justify-content: space-betweenは、アイテム間のスペースを均等に配置します。align-items: centerは、アイテムを垂直方向に中央揃えします。.itemのflex: 1`は、アイテムがコンテナの利用可能なスペースを均等に分配することを指示します。

このようにFlexboxを活用することで、レイアウトの柔軟性やスペースの効率的な利用が可能になります。
