7. CSSアニメーションとトランジション

アニメーションとトランジションの違い

  • トランジション（Transition）: 要素の状態が変化したときに、その変化を滑らかにする効果を与えるものです。例えば、ホバー時に色が変わるボタンなどで使用されます。
  • アニメーション（Animation）: 複数のキーフレームを定義し、それらの間を移動することで、要素に複雑な動きや変化を与えるものです。例えば、フェードインや回転などの効果を実装するのに使用されます。

CSSアニメーションの基本構文と例

```css @keyframes animationName { 0% { /* アニメーションの初期状態 / transform: scale(1); } 50% { / アニメーションの途中の状態 / transform: scale(1.5); } 100% { / アニメーションの最終状態 */ transform: scale(1); } }

.element { animation: animationName 2s ease infinite; } `` この例では、@keyframesを使用してanimationNameという名前のアニメーションを定義しています。そして、.element`にそのアニメーションを適用し、2秒間かけて要素を拡大縮小させています。

CSSトランジションの基本構文と例

```css .element { transition: property duration timing-function delay; }

/* 例 */ .button { transition: background-color 0.3s ease; }

.button:hover { background-color: #ff0000; } `` この例では、.buttonにbackground-colorのトランジションを設定し、ホバー時に0.3秒かけて背景色を変更する効果を与えています。ease`は変化がゆっくり始まり、終わりが速くなるようなタイミング関数です。

CSSアニメーションとトランジションを使うことで、ウェブページに動きや効果を追加し、ユーザーエクスペリエンスを向上させることができます。
