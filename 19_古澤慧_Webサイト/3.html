3. CSSの基礎

CSSとは何か

CSS（Cascading Style Sheets）は、HTMLやXMLなどのマークアップ言語で記述された文書のスタイルやレイアウトを指定するためのスタイルシート言語です。HTMLが文書の構造を定義するのに対し、CSSはその見栄えやスタイルを定義します。

セレクタ、プロパティ、値の説明

  • セレクタ（Selector）: スタイルが適用される要素を選択するためのパターンです。要素のタイプ、クラス、IDなどを指定することができます。
  • プロパティ（Property）: スタイルの特定の属性を指定します。例えば、colorやfont-sizeなどがあります。
  • 値（Value）: プロパティに割り当てる具体的な値です。色、サイズ、位置など、プロパティによって許容される値は異なります。

CSSの基本構文

CSSは、セレクタとそのスタイルの宣言の組み合わせで構成されます。例えば、次のような形式です： css セレクタ { プロパティ: 値; プロパティ: 値; ... } このようにして、特定のセレクタに対してスタイルを指定します。
